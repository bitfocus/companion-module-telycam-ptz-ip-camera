- **核心入口**：`src/main.ts` 定义了 `ModuleInstance` 类，继承自 `@companion-module/base` 的 `InstanceBase`，负责管理 UDP Socket 连接、HTTP 轮询定时器以及各功能模块的初始化。
- **通信层设计**：采用双协议混合架构。运动控制（云台、变焦、预置位）通过 `node:dgram` 发送 VISCA 十六进制指令；状态同步与图像设置（白平衡、曝光等）通过 `fetch` API 调用摄像机 HTTP CGI 接口。
- **功能模块化**：业务逻辑按 Companion 规范拆分为独立文件：`actions.ts`（动作定义）、`feedbacks.ts`（反馈逻辑）、`presets.ts`（按键预设）、`variables.ts`（变量定义）和 `config.ts`（配置表单）。
- **构建与清单**：`companion/manifest.json` 声明模块元数据及 Node.js 22 运行时要求；`package.json` 配置了基于 TypeScript 的编译流程 (`tsc`) 和 Companion 打包工具。
